What is it?

NLPTextDungeon is a text-based dungeon focused on exploration. Play the game? Have fun!

What makes it interesting:

NLPTextDungeon is built around a text interface. The player types the action they want to take in and the program attempts to interpret the statement as if it were a dungeon master. The intent is to allow for an open-ended exploration experience. See help for more info.
