Hello, welcome to the login/sign-up page.
For now, this is a VERY simplified login. If you're signed up already just enter your username and password. If you haven't registered yet just enter your desired username and password. If you get a message about a bad password, it means someone already is using that username.
Important Note on Passwords: Passwords are stored unhashed in a simple text file that's very vulnerable to attack. I highly recommend using a simple password like "123456" that you don't use elsewhere.
